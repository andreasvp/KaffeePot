# KaffeePot
Einfache Anwendung zum Verwalten einer Kaffeekasse mit Arduino Hardware

# Vorbedingungen

* CVS Datei (KaffeePot.csv) muss unter Windows Codiert sein (0A als \n)
* CVSFile Library von https://github.com/fivitti/Arduino-CSV-File
